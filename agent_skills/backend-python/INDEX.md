## ¿Cuándo se activa este pack?

Este skill se activa cuando el usuario:
- Trabaja con backend en Python (FastAPI, Django, Flask)
- Necesita patrones de diseño backend: Repository, Unit of Work, CQRS
- Busca guidance en async/await, concurrencia, o arquitectura de servicios
- Requiere best practices de SQLAlchemy, testing, o despliegue backend

## Reglas de Comportamiento

1. **Siempre** aplica los principios de `general-dev/` antes de patrones específicos de Python
2. **Siempre** prefiere async sobre sync para I/O bound operations
3. **Nunca** expongas credenciales o secretos en código — usa variables de entorno
4. **Prioriza** type hints en todas las funciones públicas
